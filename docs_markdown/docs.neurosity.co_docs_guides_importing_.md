On this page

We support ESM, Node and the Browser:

## ESM [​](https://docs.neurosity.co/docs/guides/importing/\#esm "Direct link to ESM")

```js
import { Neurosity } from "@neurosity/sdk";
```

## Node [​](https://docs.neurosity.co/docs/guides/importing/\#node "Direct link to Node")

```js
const { Neurosity } = require("@neurosity/sdk");
```

## Browser [​](https://docs.neurosity.co/docs/guides/importing/\#browser "Direct link to Browser")

```html
<script type="module">
  import { Neurosity } from "./node_modules/@neurosity/sdk/index.mjs";
</script>
<script nomodule src="./node_modules/@neurosity/sdk/neurosity.umd.js">
```
